O Que é HTML?

A Linguagem de Marcação de Hipertexto também conhecida como HTML, é uma das linguagens que compõe a maioria das páginas da internet e aplicativos.

Com ele nós podemos desenvolver vários usos, seja criando seções, parágrafos, tags e entre outros elementos.

Modelos que utilizam o HTML em sua formatação:

  • Páginas Web
  • Aplicativos
  • Documentos

O Que é CSS?

O CSS, também conhecido como Cascading Style Sheets é uma linguagem de estilização. Portanto, fontes, cores e efeitos serão atribuídos. O CSS é responsável pela decoração do site, como uma maquiagem.

E o JavaScript?

O JavaScript entra no final, sendo o responsável pelas funcionalidades e comportamento do site. Sabe aqueles menus que abrem e fecham quando você clica? Então, esse é um bom exemplo da utilização do JavaScript.
